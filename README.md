🌦️ Weather Forecast Web App

A responsive and user-friendly weather forecasting web application built using HTML, CSS, and JavaScript. The application fetches real-time weather data using the OpenWeatherMap API, allowing users to search for weather conditions in cities worldwide.

🚀 Features
🔍 Search weather information by city name
🌡️ Display current temperature and weather conditions
💧 Show humidity levels and wind speed
☁️ Dynamic weather icons based on current conditions
📱 Fully responsive design for desktop and mobile devices
⚡ Real-time weather updates through API integration

🛠️ Technologies Used
HTML5 – Structure of the application
CSS3 – Styling and responsive design
JavaScript (ES6) – Application logic and API handling
OpenWeatherMap API – Real-time weather data
